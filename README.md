# Vietnamese_Document_Scanner

## Install
```bash
    git clone
    cd Vietnamese_Document_Scanner
    
    # install dependencies
    pip install -r requirements.txt
```

## QuickStart
```commandline
    python main.py --image_path=path_to_image
```

#### visualize results
```commandline
    python main.py --image_path=path_to_image --visualize
```
